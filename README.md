# ☕ Java Fundamentals Project

This project was created to practice core Java concepts through small and practical implementations.

---

## 🎯 Objective

The goal of this repository is to improve my understanding of Java fundamentals by building a simple library management system. Through this project, I aim to model real-world concepts such as books, availability, and borrowing logic while applying object-oriented programming principles in practice.

---

## 🧠 Focus Areas

- Object-Oriented Programming (OOP)
- Encapsulation
- Code organization

---

## 🛠️ Technologies

- Java 17+
- Apache Maven
- Visual Studio Code

---

## 🚀 Features

This repository may include:

- Simple console-based applications
- Small systems (e.g., library, task manager, etc.)
- Experiments with object-oriented design
